## 이게 뭔가

시중 교재를 통째로 옮긴 사이트가 **아닙니다.**

정보보안기사 준비하면서 기출을 풀고,  
“이건 무조건 또 나온다” 싶은 것만 골라 **손글씨로 압축한 개인 요약 노트**입니다.  
빨간 펜·파란 펜 강조까지 원본 그대로예요.

PDF만 켜 두면 진도가 안 보이고,  
어느 장을 외웠는지 까먹어서… 그냥 웹으로 넘기며 체크하게 만들었습니다.

- 원본 필기 **82페이지**
- 과목·테마별로 묶어 둠 (시스템 / 네트워크 / VPN·장비 / 웹 / 실무 / 관리 / 법·개인정보 / 복습)
- 페이지마다 **암기 완료** 표시 (브라우저에만 저장, 로그인 없음)
- 약한 페이지만 **랜덤**으로 다시 뽑기

완벽하진 않아요.  
내가 공부하려고 만든 거라, 이상하면 고치면서 씁니다.

## 쓰는 법

1. 위 링크로 들어가기  
2. 과목 고르기  
3. 페이지 넘기면서 손으로 쓴 표현 그대로 외우기  
4. 됐다 싶으면 **암기 완료**  
5. 불안한 날엔 **랜덤**으로만 돌리기  

진도는 같은 PC·같은 브라우저에 남습니다.  
다른 기기면 처음부터예요. 그게 더 가볍더라고요.
